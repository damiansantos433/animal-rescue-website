# Animal Rescue Website

This project is a responsive website designed for an animal rescue organization. It provides information about the organization, its mission, and ways users can get involved.

## Live Demo
http://cis190a.nupurple.net/dsantos/rescue/index.html

## Features
- Multi-page website with clear navigation  
- Structured and user-friendly layout  
- Informational sections about services and mission  
- Basic responsive design  

## Technologies Used
- HTML  
- CSS  
- JavaScript   

## What I Learned
- Structuring and organizing multi-page websites  
- Designing clean and readable layouts  
- Applying CSS for styling and responsiveness  
- Improving overall user experience  

## How to Run Locally
1. Clone or download the repository  
2. Open `index.html` in your browser  

## Notes
This project highlights my early experience in frontend development, with a focus on clean layout design, navigation, and overall user experience.
